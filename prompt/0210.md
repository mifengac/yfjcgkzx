这是一份为您定制的**通用拆分脚本**。

它的核心功能是：**读取 Excel -> 锁定指定列 -> 识别特定关键词 -> 将该单元格内的“多人名单”拆解为“多行数据”，并自动生成序号**。

### 脚本说明
1.  **通用性**：你只需要修改脚本最上方的 `CONFIG` 配置区域（文件名、要分析哪一列、关键词是什么），就可以处理不同批次的数据。
2.  **自动保留**：它会保留原始表格里的所有其他信息（如时间、地点、经纬度），并在后面追加拆分出来的人员信息。
3.  **序号逻辑**：同一个单元格拆出来的人，共用同一个`序号`，方便你按组管理。

### Python 脚本 (`split_tool.py`)

请确保你的内网电脑安装了 `pandas` 和 `openpyxl` 库。

```python
import pandas as pd
import re
import os
from datetime import datetime

# ================= 脚本配置区域 (请根据实际情况修改) =================

# 1. 输入的文件名 (必须是 .xlsx 格式)
INPUT_FILE = 'source.xlsx'

# 2. 输出的文件名 (会自动加上时间戳)
OUTPUT_FILE = f'人员拆分结果_{datetime.now().strftime("%Y%m%d_%H%M")}.xlsx'

# 3. 指定包含聚集信息的列名 (Excel表头里的名字)
# 例如你的表头是 "研判详情" 或 "聚集信息"，请在这里修改
TARGET_COLUMN = '研判详情' 

# 4. 指定拆分的关键词 (程序会从这个词后面开始截取数据)
# 例如数据是 "...聚集学生人员为：张三..."，这里就填 "聚集学生人员为："
# 如果你的数据有时候是 "聚集学生为：", 可以只填 "为："，但建议填准确点防止误判
SPLIT_KEYWORD = '聚集学生人员为：'

# ===================================================================

def extract_people(text, keyword):
    """
    从文本中提取人员信息的通用逻辑
    """
    if not isinstance(text, str) or keyword not in text:
        return []
    
    # 1. 截取关键词后面的内容
    # 比如 "xxx...聚集学生人员为：133(ID、名)；134(ID、名)..."
    # 我们只取冒号后面的部分
    content_after_keyword = text.split(keyword)[1]
    
    # 2. 正则表达式匹配逻辑
    # 匹配模式：手机号 + 全角/半角括号 + (身份证号 + 顿号/逗号 + 姓名) + 全角/半角括号
    # \d{11} : 11位手机号
    # [（\(] : 中文括号或英文括号
    # (.*?) : 身份证号 (兼容18位或不规则)
    # [、,，] : 分隔符 (兼容顿号、逗号)
    # (.*?) : 姓名
    # [）\)] : 闭合括号
    
    pattern = re.compile(r'(\d{11})[（\(](.*?)[、,，](.*?)[）\)]')
    
    # findall 返回列表: [('133...', '445...', '张三'), ('134...', '...', '李四')]
    people_list = pattern.findall(content_after_keyword)
    
    return people_list

def main():
    print("================ 通用人员拆分工具 ================")
    print(f"读取文件: {INPUT_FILE}")
    print(f"目标列名: {TARGET_COLUMN}")
    print(f"拆分关键词: {SPLIT_KEYWORD}")
    print("------------------------------------------------")

    # 1. 读取数据
    if not os.path.exists(INPUT_FILE):
        print(f"错误: 找不到文件 {INPUT_FILE}，请确认文件名是否正确。")
        return

    try:
        df = pd.read_excel(INPUT_FILE)
    except Exception as e:
        print(f"读取 Excel 失败: {e}")
        return

    # 检查目标列是否存在
    if TARGET_COLUMN not in df.columns:
        print(f"错误: 在表格中找不到列名 '{TARGET_COLUMN}'。")
        print(f"当前表格的列名有: {list(df.columns)}")
        return

    # 2. 开始处理
    new_rows = []
    group_serial = 1 # 团伙序号
    
    success_count = 0
    
    print("正在处理数据...")

    for index, row in df.iterrows():
        cell_content = row[TARGET_COLUMN]
        
        # 提取人员列表
        people = extract_people(cell_content, SPLIT_KEYWORD)
        
        if people:
            # 如果提取到了人，就进行裂变（一行变多行）
            for person in people:
                phone = person[0]
                id_card = person[1]
                name = person[2]
                
                # 复制原始行的所有数据
                new_row = row.to_dict()
                
                # 添加新的拆分字段
                new_row['序号'] = group_serial
                new_row['拆分_姓名'] = name.strip()
                new_row['拆分_身份证号'] = id_card.strip()
                new_row['拆分_手机号'] = phone.strip()
                
                new_rows.append(new_row)
            
            group_serial += 1
            success_count += 1
        else:
            # 可选：如果这一行没有找到人员名单，是否要保留？
            # 方案A：保留原始行，但拆分字段为空 (取消注释下面代码即可保留)
            # new_row = row.to_dict()
            # new_row['序号'] = '无'
            # new_rows.append(new_row)
            pass

    # 3. 转换为 DataFrame
    result_df = pd.DataFrame(new_rows)

    if result_df.empty:
        print("警告: 没有拆分出任何数据。请检查‘目标列名’和‘拆分关键词’是否正确。")
        return

    # 4. 调整列顺序 (把拆分出来的关键列放到最前面，方便查看)
    # 获取所有列名
    cols = list(result_df.columns)
    # 定义想要优先显示的列
    priority_cols = ['序号', '拆分_姓名', '拆分_身份证号', '拆分_手机号']
    # 移除这些列在原列表的位置
    for c in priority_cols:
        if c in cols:
            cols.remove(c)
    # 组合新顺序
    final_cols = priority_cols + cols
    result_df = result_df[final_cols]

    # 5. 保存结果
    try:
        result_df.to_excel(OUTPUT_FILE, index=False)
        print("------------------------------------------------")
        print(f"处理完成！")
        print(f"共发现 {success_count} 个聚集组，拆解出 {len(result_df)} 人。")
        print(f"结果已保存为: {OUTPUT_FILE}")
    except Exception as e:
        print(f"保存文件失败: {e}")

if __name__ == '__main__':
    main()
```

### 怎么使用这个通用脚本？

1.  **准备数据**：将你要处理的 Excel 文件放在脚本同目录下，重命名为 `source.xlsx`（或者在代码里改名）。
2.  **确认列名**：打开你的 Excel，看一眼包含那一大串文本的表头叫什么名字（比如叫`研判结果`）。
3.  **确认关键词**：看一眼那一列里，人名名单是从哪个词开始的（比如`聚集学生人员为：`）。
4.  **修改代码**：用记事本打开脚本，修改最上面的 `TARGET_COLUMN` 和 `SPLIT_KEYWORD`。
5.  **运行**：双击运行或命令行运行。
