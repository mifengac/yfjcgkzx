**使用示例:**

```sql
SELECT * 
FROM ywdata.v_jq_optimized 
WHERE code_type = 'original' 
  AND leixing = '打架斗殴' 
  AND calltime BETWEEN '2026-01-01' AND '2026-02-06';

SELECT * 
FROM ywdata.v_jq_optimized 
WHERE code_type = 'confirmed' 
  AND leixing = '打架斗殴' 
  AND calltime BETWEEN '2026-01-01' AND '2026-02-06';

-- 查询所有匹配的记录(原始+确认)
SELECT * 
FROM ywdata.v_jq_optimized 
WHERE leixing = '打架斗殴' 
  AND calltime BETWEEN '2026-01-01' AND '2026-02-06';

SELECT DISTINCT ON (id) *
FROM ywdata.v_jq_optimized 
WHERE leixing = '打架斗殴' 
  AND calltime BETWEEN '2026-01-01' AND '2026-02-06';
```

现在您可以通过 `code_type` 字段灵活控制要查询原始警情还是确认警情的匹配结果了。
