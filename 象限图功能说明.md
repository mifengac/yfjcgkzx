# 巡防统计模块 - 导出象限图功能说明

## 功能概述
在巡防统计模块新增"导出象限图"功能，用于对全市派出所的"巡防投入"和"警情风险"进行对比分析，生成象限图Excel文件。

## 实现的功能

### 1. 前端界面
- 在巡防统计页面添加了"导出象限图"按钮
- 按钮样式：橙色背景，与其他按钮保持一致
- 点击按钮会调用导出象限图功能

### 2. 后端路由
- 新增路由：`/xunfang/export_quadrant_chart`
- 支持POST请求，接收开始时间和结束时间参数
- 返回Excel文件下载

### 3. 业务逻辑

#### 3.1 获取全市派出所数据
```sql
SELECT dq."parent_code", dq."code", dq."name" 
FROM "sys_dq_pcs" dq 
WHERE dq."level"=2 and dq."name" ~~ '%派出所%'
```

#### 3.2 计算巡防投入指数（横坐标）
根据不同的巡逻类型计算权重值：
- **车巡** (deploymentType=1): 权重 1.3
  - 巡逻时长(小时) + 巡逻里程(公里) × 1.3
- **步巡** (deploymentType=2): 权重 1.0
  - 巡逻时长(小时) + 巡逻里程(公里) × 1.0
- **摩托车巡** (deploymentType=3): 权重 1.3
  - 巡逻时长(小时) + 巡逻里程(公里) × 1.3
- **车巡+步巡** (deploymentType=15): 权重 1.5
  - 巡逻时长(小时) + 巡逻里程(公里) × 1.5

#### 3.3 计算警情风险指数（纵坐标）
根据不同案件类型计算权重值：
- **人身伤害类**: 权重 1.5
- **侵犯财产类**: 权重 1.2
- **扰乱秩序类**: 权重 1.0

#### 3.4 数据来源
- **巡防数据**: 通过 `get_cross_day_data` 函数获取，与导出在线在岗功能相同
- **警情数据**: 从 `zq_kshddpt_dsjfx_jq` 表中按时间范围和案件类型查询

### 4. Excel文件生成
- **工作表名称**: "派出所巡防警情象限图"
- **数据列**: 派出所名称、派出所代码、所属地区代码、巡防投入指数、警情风险指数
- **图表类型**: 散点图（象限图）
- **统计信息**: 包含平均值和总数统计

## 使用方法

1. 进入巡防统计页面
2. 选择开始时间和结束时间
3. 点击"导出象限图"按钮
4. 系统会自动计算并下载Excel文件
5. Excel文件包含：
   - 派出所数据表格
   - 象限散点图
   - 统计信息

## 技术特点

1. **数据准确性**: 确保只计算派出所级别的数据
2. **权重计算**: 按照业务需求设置不同类型权重
3. **图表可视化**: 使用散点图直观展示巡防投入与警情风险关系
4. **错误处理**: 完善的异常处理和日志记录
5. **性能优化**: 合理的数据库查询和数据处理
6. **智能标签布局**: 自动调整数据点标签位置，避免重叠
7. **详细数据导出**: 同时提供象限图PNG和原始数据Excel文件
8. **多层数据展示**: Excel包含巡防原始数据、警情原始数据和汇总数据

## 最新功能改进（2025-10-10）

### 1. 权重调整
- 车巡权重从1.2调整为1.3，与摩托车巡保持一致

### 2. 智能标签位置调整
- 实现了基于数据点密度的智能标签位置算法
- 根据象限位置和标签长度自动选择最佳显示位置
- 添加连线指示标签对应的数据点
- 有效避免了标签重叠问题

### 3. 增强Excel导出功能
导出的ZIP文件包含：
- **PNG象限图**: 高质量的matplotlib象限散点图
- **Excel工作簿**: 包含三个工作表
  - 巡防原始数据：各类巡逻的时长、里程等详细数据
  - 警情原始数据：各类案件的具体数量
  - 汇总数据：各派出所的最终指数和统计信息

### 4. 文件导出方式
- 改为ZIP压缩包导出，同时包含PNG图片和Excel文件
- 文件命名：`{开始时间}至{结束时间}派出所巡防警情象限图.zip`
- 前端提示：明确告知用户导出内容包含象限图和原始数据
